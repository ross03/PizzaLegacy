# Winner Images Folder

## How to Add Token Images

For each tournament, you need to add THREE images to this folder:

### File Naming Convention:
- `YYYY-MM-DD-winner.png` - Winner token
- `YYYY-MM-DD-top8.png` - Top 8 promo
- `YYYY-MM-DD-participation.png` - Participation promo

### Example:
For a tournament on April 18, 2025:
```
2025-04-18-winner.png
2025-04-18-top8.png
2025-04-18-participation.png
```

### Important:
- Date format MUST be YYYY-MM-DD (year-month-day)
- Use dashes (-) not underscores
- File extension should be .png (lowercase)
- The date must match the date you enter in config/config.js

### Current Tournament Dates:
Refer to config/config.js for the list of all tournament dates.

When you add new images here, remember to also update the winners array in config/config.js!
